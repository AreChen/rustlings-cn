## 快速开始

```bash
# 从我们的 fork 安装中文版 CLI
cargo install --git https://github.com/AreChen/rustlings-cn.git --branch main --locked --force
# 创建独立的学习目录
mkdir rustlings-cn-study
cd rustlings-cn-study
rustlings init
# 进入新目录
cd rustlings
# 启动 Rustlings
rustlings
```

更多细节请参阅[**安装**](@/setup/index.md)页面 🧰
