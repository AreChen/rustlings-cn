# 枚举

Rust 允许你定义称为“枚举”的类型，用来列举所有可能的值。
许多语言都提供枚举，但它们在不同语言中的能力各不相同。Rust 的枚举与函数式语言
（例如 F#、OCaml 和 Haskell）中的代数数据类型最为相似。
枚举通常与 Rust 的“模式匹配”机制结合使用，这使得针对枚举的不同值运行不同代码变得很容易。
