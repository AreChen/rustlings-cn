# 错误处理

大多数错误并不严重到需要程序完全停止。
有时函数失败的原因很容易理解，也很容易采取相应措施。
例如，如果你尝试打开一个文件，但因为文件不存在而失败，可能会希望创建该文件，
而不是终止进程。
