# 哈希映射

*哈希映射* 可以将一个值与特定的键关联起来。
在 C++ 中它也被称为 [*无序映射*](https://en.cppreference.com/w/cpp/container/unordered_map)，
在 Python 中被称为 [*字典*](https://docs.python.org/3/tutorial/datastructures.html#dictionaries)，
在其他语言中则可能被称为 *关联数组*。

这是我们之前讨论向量时提到的另一种数据结构。
